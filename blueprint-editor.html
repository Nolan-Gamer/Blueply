Types de blocs

Event
Déclenche la logique. Start s'exécute une fois, Update à chaque frame.
Action
Fait quelque chose : bouger, sauter, tourner…
Condition
Vérifie si quelque chose est vrai (touche pressée, personnage au sol…).
Debug
Affiche des messages dans la console Unity pour tester.

Utiliser le script dans Unity

1
Ouvre Unity et crée un nouveau projet 3D (ou 2D).
2
Dans la fenêtre Project, fais clic droit → Create → Folder. Nomme-le Scripts.
3
Copie le fichier PlayerController.cs généré dans ce dossier Assets/Scripts/.
4
Dans la Hierarchy, sélectionne ton GameObject (ex: Player).
5
Glisse le script depuis le dossier Scripts sur l'Inspector du GameObject. Il est maintenant attaché !
6
Appuie sur ▶ Play dans Unity pour tester.

Personnaliser le code

✏️
Vitesse : cherche moveSpeed et change le nombre.
✏️
Saut : cherche jumpForce et change le nombre.
✏️
Touches : cherche KeyCode.Space et change la touche.
✏️
Nom personnage : modifie playerName en haut du script.

Composants Unity requis

⚙️
Ajoute un Rigidbody à ton Player pour la physique (saut, gravité).
⚙️
Ajoute un Collider (Box ou Capsule) pour les collisions.
